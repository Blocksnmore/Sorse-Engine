    Draw methods

    Sorse Engine has multiple easy to use methods to draw to the screen

    drawRect(options)

    Draw a Rectangle

    1
2
3
4
5
6
    sorse.drawRect({
    x: 0,
    y: 0,
    width: 100,
    height: 100,
});

    drawCircle(options)

    Draw a Circle

    1
2
3
4
5
    sorse.drawCircle({
    x: 100,
    y: 100,
    radius: 50,
});

    drawImage(options)

    Draw an Image

    1
2
3
4
5
6
7
    sorse.drawImage({
    image: './sorse.png',
    x: 0,
    y: 0,
    width: 100,
    height: 100,
});

    drawText(options)

    Draw some Text

    1
2
3
4
5
6
    sorse.drawText({
    text: 'Hello from Sorse Engine!',
    x: 100,
    y: 100,
    size: 24,
});

    drawLine(options)

    Draw a Line

    1
2
3
4
5
6
7
    sorse.drawLine({
    x1: 0,
    y1: 0,
    x2: 100,
    y2: 100,
    size: 10,
});

    drawOval(options)

    Draw an Oval

    1
2
3
4
5
6
7
    sorse.drawOval({
    x: 100,
    y: 100,
    radiusX: 10,
    radiusY: 5,
    rotation: 0,
});

    Clear the entire screen

    sorse.clearScreen();
